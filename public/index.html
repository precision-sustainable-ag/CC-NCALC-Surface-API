Cover Crop Nitrogen Calculator (CC-NCALC)

Data API

Parameter Units Description Default Model
lat degrees Latitude required both
lon degrees Longitude required both
start yyyy-mm-dd Start date required both
end yyyy-mm-dd Ending date required both
n % Nitrogen in cover crop required both
biomass kg/ha Dry biomass required both
lwc g water/
g dry biomass		 Cover Crop Water Content at Termination required surface
carb % Carbohydrates in cover crop 24.7 + 10.5n both
cell % Hemicellulose in cover crop 69 - 10.2n both
lign % Lignin in cover crop 100 - (carb + cell) both
om % Soil organic matter SSURGO API both
bd g/cm3 Bulk density SSURGO API both
in ppm Soil inorganic N 10 both
pmn mg N/kg Potentially mineralizable N 10 both

Examples

  • surface
    https://api.precisionsustainableag.org/cc-ncalc/surface?start=2020-04-26&end=2020-10-22&lat=34.88689&lon=-83.41941&n=10&biomass=3000&lwc=10

Notes

  • By default, the JSON output is an array of objects.
    You can output an object of arrays by using the parameter: layout=object
    Example: https://api.precisionsustainableag.org/cc-ncalc/surface?start=2020-04-26&end=2020-10-22&lat=34.88689&lon=-83.41941&n=10&biomass=3000&lwc=10&layout=object
  • You can output as CSV instead of JSON by including:
    &output=csv
  • Weather data is cached per lat/lon and start/end.
